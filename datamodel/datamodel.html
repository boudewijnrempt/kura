lng_element

Description

Analyzed part of speech - a word, a morpheme, a phrase, perhaps even a phoneme.
Recusively decomposable:

element 1 eseri glosse: 'your'
element 1.1 e- POSS (possessive prefix)
element 1.2 seri 2smHGH (second person masculine high grade honorific)

lng_project

Description

Projects order data into conceptual entities, like 'A Grammar of Denden', or 'Second person pronouns in Valdyan'. A project can span more than one language. More than one user can work on a project. A project can be supported by grants and have a homepage where the results of the project can be published to.

lng_tagtypecode

Description

Tags can be divided into several types. The type of a tag determines functionality. For instance:

glosse - a translation that can be used in the interlinear presentation of a text
translation: a longer translation
note: grammatical or antropological notes
topic: the ordeirng of the data for use in active grammars
link: network link to more data
